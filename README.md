# bank-loan-analysis
#Project Overview:
This project analyzes a bank loan dataset to identify the factors that influence loan approval and loan default. The analysis includes data cleaning, SQL exploration, Python visualizations, and interactive dashboards in Power BI and Excel.

#🧰 Built With
Python (Pandas) — cleaned messy real-world-style data: fixed inconsistent labels, handled nulls, engineered features like income ratios.
MySQL — wrote grouped, filtered, and segmented queries to answer specific risk questions.
Matplotlib & Seaborn — built a 6-panel visual EDA dashboard, including a correlation heatmap and trend lines.
Power BI — built an interactive dashboard with DAX measures, a custom calendar table, and cross-filtering slicers.
Excel — built a PivotTable-powered dashboard for a lighter, no-code-required view of the same data.

#📁 How This Repo Is Organized
data/ — raw and cleaned datasets.
notebooks/ — cleaning → SQL analysis → visualization, in that order.
images/ — exported chart images.
power_bi/ — the .pbix dashboard file.
excel_dashboard/ — the Excel workbook dashboard.

#🔑 Key Insights
*Credit history is the strongest predictor of default.
*Income predicts loan size, not repayment behavior.
*Interest rate shows no real link to loan amount.
*Home loans dominate the portfolio.
*Urban and Semiurban areas carry most loan volume.
Loan volume stays steady across months, no major seasonality
